# 📰 Spam News Detector (Machine Learning Project)

## 📌 Project Overview

Spam News Detector is a Machine Learning based web application that identifies whether a news article is **Real** or **Fake/Spam**.
The system uses Natural Language Processing (NLP) techniques and a trained ML model to analyze the text and classify the news.

This project demonstrates the practical implementation of Machine Learning concepts such as **text preprocessing, TF-IDF vectorization, and classification models**.

---

## 🚀 Features

* Detects fake or spam news from text input
* Machine Learning based classification
* Web interface built using Flask
* Fast predictions using trained model
* Clean and simple UI

---

## 🧠 Machine Learning Model

The project uses:

* Logistic Regression
* TF-IDF Vectorizer
* Text preprocessing techniques

Dataset used:

* Fake and Real News Dataset (~44,000 articles)

Model accuracy:

* Approximately **93–96%**

http://127.0.0.1:5000
